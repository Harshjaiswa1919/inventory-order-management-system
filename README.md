# Inventory Order Management System

A full-stack Inventory Management System built using FastAPI, React.js, PostgreSQL, and SQLAlchemy.

## Features

* Product Management (Add, View, Delete)
* Customer Management (Add, View, Delete)
* Order Management (Create, View, Delete)
* Automatic Stock Reduction
* Dashboard Analytics
* Inventory Tracking

## Technology Stack

### Backend

* FastAPI
* SQLAlchemy
* PostgreSQL

### Frontend

* React.js
* Axios

### Database

* PostgreSQL

## Project Modules

1. Products Module
2. Customers Module
3. Orders Module
4. Dashboard Module
